# ForTests

Android app for tests.

## General info

ForTests is a student project.

## Technologies

* Android studio 4.0
* Android SDK 23
* JUnit 4.12
* Mockito 1.10.19

## Setup

To setup a project, you need to clone the repository or download it's content and open up the main folder in Android Studio.
